title:: 1798. Maximum Number of Consecutive Values You Can Make

- ## Description
-
- ## Solution
	- ```java
	  // time:  O(logN)
	  // space: O(1)
	  class Solution {
	      public int getMaximumConsecutive(int[] coins) {
	          Arrays.sort(coins);
	          int res = 0;
	          for (int coin : coins) { 
	              // coins is sorted, no smaller 
	              // res + 1 is the next consecutive sum
	              if (coin > res + 1) {
	                  break;
	              }
	              res += coin;
	          }
	          return res + 1; // add 0 to result
	      }
	  }
	  ```
